# 書房系統設計

*建立日期：2026-02-11*

---

## 一、設計目標

1. **取代商店**：提供構築最佳化的場所，但以「學習」而非「購買」為核心
2. **Deck Thinning**：透過「暫放」機制實現牌組精簡，不違背學習理念
3. **策略深度**：學習點數（LP）作為貨幣，讓答對率直接轉化為構築選項
4. **詞族進化入口**：書房是進化和深化的主要場所之一

---

## 二、核心貨幣：學習點數 (LP)

### 獲取方式

| 來源 | LP |
|------|-----|
| 戰鬥中答對 | +2 |
| 答對且速度快（< 3 秒） | +3 |
| 完美通關（該場戰鬥全答對） | +5 bonus |
| 進化卡牌時答對新字 | +1 |

### 設計意圖

- 答對率越高 → LP 越多 → 書房選擇越豐富 → 牌組越精準 → 戰鬥越強
- 形成「學得好 → 更多選擇 → 更強」的正循環
- LP 不跨 run 保留，每次 run 從 0 開始

---

## 三、書房服務

| 服務 | 費用 | 效果 | 策略用途 |
|------|------|------|---------|
| **預習** | 8 LP | 從未學池自由選擇 1 張想學的字（不限 3 選 1） | 精準構築，挑選特定維度/元素的字 |
| **暫放** | 5 LP | 將 1 張卡從本次 run 移出（學習紀錄保留） | Deck thinning，聚焦核心卡牌 |
| **進化** | 0 LP | 對 Lv.5+ 卡進行詞族進化（詳見進化系統文檔） | 詞族系統入口，擴展卡池 |
| **深化** | 6 LP | 對 Lv.5+ 卡提升等級上限至 Lv.9 | 極致專精路線 |
| **筆記** | 3 LP | 購買消耗品（見下方） | 戰術輔助 |

### 消耗品（筆記）

| 名稱 | 效果 | 持續 |
|------|------|------|
| **提示紙條** | 下次答題時顯示第一個字母 | 單次 |
| **延時沙漏** | 下次答題時間 +5 秒 | 單次 |
| **豁免令** | 下次答題可跳過不扣分（但也不得分） | 單次 |

消耗品最多攜帶 3 個，不跨 run 保留。

## 四、「暫放」機制詳解

### 與殺戮尖塔「移除卡牌」的比較

| | 殺戮尖塔 | 本遊戲 |
|--|---------|--------|
| 名稱 | 移除卡牌 | 暫放卡牌 |
| 效果 | 卡牌永久從牌組消失 | 卡牌從本次 run 移出 |
| 學習影響 | 無 | 學習紀錄保留，熟練度不受影響 |
| 包裝 | 「商人幫你銷毀」 | 「暫時放回書架，這次冒險不帶」 |
| 策略意義 | 精簡牌組、移除弱卡 | 聚焦複習、優化出牌密度 |

### 暫放的限制

- 每次書房訪問最多暫放 **1 張**
- 牌組最少保留 **5 張卡**（不能暫放到低於此數）
- 起始牌組中的基礎卡可以暫放（但基礎卡通常效果弱，是暫放的優先目標）

### 暫放的策略價值

精簡牌組是 roguelike 卡牌遊戲的核心策略之一：
- 牌組越小 → 關鍵卡的抽到率越高 → 協同更穩定觸發
- 暫放弱卡/不符合構築方向的卡 → 提升整體出牌品質
- 但暫放也意味著失去該卡的維度/元素貢獻 → 影響湧現式協同

## 五、書房 vs 休息站的定位區分

| | 休息站 | 書房 |
|--|--------|------|
| **主題** | 休息與修煉 | 研究與選擇 |
| **核心功能** | 回血/升級(答題)/學新字 | 預習/暫放/進化/深化/筆記 |
| **貨幣** | 機會成本（三選一） | 學習點數 (LP) |
| **每章出現** | 2-3 次（含 Boss 前必出 1 次） | 1-2 次 |
| **策略定位** | 生存 vs 成長 | 構築最佳化 |
| **氣氛** | 營火旁的休息 | 燈下的研究 |

## 六、地圖整合

書房作為第七種房間類型加入地圖系統：

| 房間 | 圖示 | 功能 |
|------|------|------|
| 敵人 | ⚔️ | 普通戰鬥 |
| 精英 | 💀 | 強敵戰鬥 |
| 休息站 | 🏕️ | 三選一 |
| 寶箱 | 🎁 | 遺物/藥水 |
| 未知 | ❓ | 隨機事件 |
| **書房** | 📚 | 構築最佳化 |
| Boss | 👑 | 章節 Boss |

### 書房的地圖規則

- 每章出現 1-2 次
- 不出現在前 3 步（太早沒有足夠 LP）
- 不與休息站相鄰（避免連續非戰鬥節奏過緩）
- 建議出現在第 6-8 步和第 11-13 步（上下半場各一次）

### 更新後的房間分佈比例（每 15 步）

| 房間類型 | 數量 | 比例 |
|---------|------|------|
| 敵人 | 5-6 | ~37% |
| 精英 | 2 | ~13% |
| 休息站 | 2（含 Boss 前） | ~13% |
| 寶箱 | 1（第 9 步固定） | ~7% |
| 書房 | 1-2 | ~10% |
| 未知 | 3-4 | ~20% |

## 七、書房與其他系統的互動

| 系統 | 互動方式 |
|------|---------|
| **詞族進化** | 書房是進化/深化的場所之一 |
| **遺物系統** | 書房提供遺物獲取途徑（研究服務，未來版本） |
| **湧現式協同** | 暫放影響維度/元素覆蓋率，預習可針對性補充缺失維度 |
| **學習系統** | LP 來自答題表現，書房消費驅動更認真答題 |
