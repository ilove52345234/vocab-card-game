# Vocab Card Game v2 — 核心設計文件

> 基於 Balatro 設計分析的全新設計，學習融入遊戲機制本身。

---

## 一、設計哲學

```
極簡的戰鬥驗證 + 複雜的 Build 構築 + 學習 = 遊戲操作本身
```

- 得分 = 籌碼 × 倍率（全遊戲只需理解這一個公式）
- 學習不是考試，而是**遊戲機制的一部分**
- 玩家以為自己在組牌型打分數，其實在認識單字、學詞性、記分類

## 二、超級概念

### 2.1 主概念：單字卡組合

- 類比撲克牌型，用單字的屬性組合出「牌型」得分
- 人人都認識英文字母，零門檻
- 「組合英文單字」比「背單字」有趣得多

### 2.2 雙維度系統

| 維度 | 類比撲克 | 數量 | 視覺呈現 |
|------|---------|------|---------|
| **元素（Element）** | 花色 | 5 種 | 卡牌底色/邊框顏色+圖示 |
| **詞性（Part of Speech）** | 點數 | 4 種 | 卡牌頂部標記 |

**五大元素：**

| 元素 | 顏色 | 圖示 | 涵蓋詞彙 |
|------|------|------|---------|
| 🌿 Life | 綠 | 葉子 | 動物、植物、食物、身體 |
| 🔥 Force | 紅 | 火焰 | 動作、運動、武器、工具 |
| 💧 Mind | 藍 | 水滴 | 情感、感官、溝通、藝術 |
| ⚙️ Matter | 灰 | 齒輪 | 物品、建築、材料、科技 |
| ✨ Abstract | 紫 | 星星 | 數字、時間、形狀、概念 |

**四大詞性：**

| 詞性 | 標記 | 佔比 |
|------|------|------|
| 名詞（Noun） | N | ~40% |
| 動詞（Verb） | V | ~25% |
| 形容詞（Adjective） | A | ~20% |
| 副詞/其他（Adverb+） | D | ~15% |

## 三、牌型系統

### 3.1 基礎規則

| 項目 | 數值 | 設計邏輯 |
|------|------|---------|
| 每關出牌次數 | 4 | 和 Balatro 一致 |
| 每關棄牌次數 | 3 | 和 Balatro 一致 |
| 手牌上限 | 8 | 9張會質變（某些牌型必定成立），所以定8 |
| 最多打出 | 5 張 | 和 Balatro 一致 |
| 初始牌組 | 52 張 | 基礎池，5元素×4詞性均勻分佈 |

### 3.2 牌型定義

| 牌型 | 條件 | 類比撲克 | 教育價值 |
|------|------|---------|---------|
| **單字 (Word)** | 打出 1 張 | 高牌 | 基礎認字 |
| **同性對 (PoS Pair)** | 2 張同詞性 | 對子 | 認識詞性 |
| **同族對 (Elem Pair)** | 2 張同元素 | - | 認識主題分類 |
| **三同性 (PoS Triple)** | 3 張同詞性 | 三條 | 強化詞性意識 |
| **語序鏈 (Grammar Chain)** | 3+ 張按語法順序 | 順子 | 學英文語序 |
| **同族三 (Elem Triple)** | 3 張同元素 | - | 主題詞彙群 |
| **滿堂 (Full House)** | 3張同詞性 + 2張同詞性 | 葫蘆 | 雙重詞性辨識 |
| **同族花 (Elem Flush)** | 5 張同元素 | 同花 | 深度主題學習 |
| **全同性 (PoS Flush)** | 5 張同詞性 | - | 詞性精通 |
| **語序同族 (Grammar Flush)** | 語序鏈 + 全同元素 | 同花順 | 終極組合 |

### 3.3 語序鏈規則

英文基本語法順序：**形容詞 → 名詞 → 動詞 → 副詞**（A → N → V → D）

合法的語序鏈範例：
- A → N → V（3張：beautiful → cat → run）
- N → V → D（3張：dog → eat → quickly）
- A → N → V → D（4張：big → house → stand → firmly）
- A → A → N → V → D（5張，重複詞性允許但必須維持順序）

規則：
- 最少 3 張
- 詞性必須按 A → N → V → D 順序排列（可跳過，如 A → V 合法）
- 同詞性可以連續出現（如 A → A → N 合法）
- 逆序不合法

### 3.4 牌型基礎分數

| 牌型 | 基礎籌碼 | 籌碼成長/級 | 基礎倍率 | 倍率成長/級 |
|------|---------|-----------|---------|-----------|
| 單字 | 5 | +10 | 1 | +1 |
| 同性對 | 10 | +15 | 2 | +1 |
| 同族對 | 15 | +15 | 2 | +1 |
| 三同性 | 30 | +20 | 3 | +2 |
| 語序鏈 | 30 | +30 | 4 | +3 |
| 同族三 | 35 | +20 | 3 | +2 |
| 滿堂 | 40 | +25 | 4 | +2 |
| 同族花 | 50 | +30 | 5 | +3 |
| 全同性 | 60 | +35 | 6 | +3 |
| 語序同族 | 100 | +40 | 8 | +4 |

> 語序鏈的倍率成長高於同族三，因為後期調整牌組成順子結構比同元素更難。

### 3.5 得分公式

```
得分 = (牌型基礎籌碼 + Σ打出卡牌的籌碼) × (牌型基礎倍率 + Σ加算倍率) × Π乘算倍率
```

每張單字卡自帶籌碼（基於字母數）：
- 3 字母：+3 籌碼
- 4 字母：+4 籌碼
- 5 字母：+5 籌碼
- 6 字母：+7 籌碼
- 7+ 字母：+10 籌碼

> 長單字更難記但給更多籌碼，內在動機驅動學習難字。

## 四、學習系統（融入局內）

### 4.1 核心原則

```
學習 = 遊戲操作本身，不是額外的考試
```

- 從來不彈出「請選擇正確答案」的測驗畫面
- 學習透過遊戲機制自然發生
- 玩家有動機學習，因為學會的卡牌更強

### 4.2 卡牌資訊揭露系統

卡牌顯示的資訊量隨學習等級增加：

| 學習等級 | 卡面顯示 | 籌碼加成 | 如何升級 |
|---------|---------|---------|---------|
| **Lv.0 未知** | 元素顏色 + [???] + 基礎籌碼 | ×0.5 | 打出此卡 3 次（看到英文+中文） |
| **Lv.1 識讀** | 英文 + 中文 + 元素 + 詞性 | ×1.0（基準） | 打出/使用此卡 10 次 |
| **Lv.2 聽力** | 同上 + 🔊 可播放發音 | ×1.2 | 在商店完成聽力強化 |
| **Lv.3 拼字** | 同上 + ✍️ 拼字標記 | ×1.5 | 在商店完成拼字強化 |
| **Lv.4 精通** | 發光特效 + 全部資訊 | ×1.5 + 可獲得版本效果 | 遺忘曲線不再退化 |

### 4.3 局內學習的發生方式

**1. 出牌動畫即學習（被動曝光）**
- 打出卡牌時，結算動畫短暫顯示：英文 → 中文 → 發音（1-2秒）
- 類似 Balatro 出牌時每張牌依序結算的動畫
- 不需要任何操作，自然記憶

**2. 組牌型即學詞性（主動認知）**
- 要組「同性對」你必須知道哪些卡是名詞、哪些是動詞
- 要組「語序鏈」你必須知道 A→N→V→D 的英文語法順序
- 要組「同族花」你必須知道哪些詞屬於同一主題
- 遊戲機制本身驅動學習

**3. Boss 盲注的模糊機制（學習=策略優勢）**
- 部分 Boss 盲注會讓卡牌資訊模糊（類似 Balatro 的「背面朝上」）
- 但已精通（Lv.4）的卡不受影響
- 越熟悉的卡 → 越不受 Boss 影響 → 學習 = 直接的策略優勢

**4. 商店的學習強化（可選，付費）**
- 商店除了買道具，還可以花金幣進行「學習強化」：
  - 🔊 **聽力練習**（$3）：播放一張卡的發音，聽完升級為 Lv.2
  - ✍️ **拼字練習**（$5）：拼對一張卡的英文，升級為 Lv.3
- 這些是花錢買的可選成長，類似 Balatro 花錢重擲商店
- 不做也能玩，但做了卡牌更強

### 4.4 遺忘曲線（局外 Meta 層）

- 基於簡化的間隔重複：Lv.1-3 的卡牌有複習間隔
- 超過間隔未使用（在任何遊戲中打出過） → 降一級
- Lv.4 精通的卡永不退化
- 複習間隔：Lv.1（1天）→ Lv.2（3天）→ Lv.3（7天）

| 等級 | 複習間隔 | 退化條件 | 退化效果 |
|------|---------|---------|---------|
| Lv.0 | - | - | - |
| Lv.1 | 1 天 | 1天內未在任何局中打出 | → Lv.0 |
| Lv.2 | 3 天 | 3天內未打出 | → Lv.1 |
| Lv.3 | 7 天 | 7天內未打出 | → Lv.2 |
| Lv.4 | 永久 | 不退化 | - |

> 退化的懲罰是籌碼降低，不是卡牌消失。玩家動機：保持卡牌強度 → 自然驅動複習。

### 4.5 新單字解鎖

- 每場遊戲開始時，從「未學習池」中自動加入 2-3 張新卡（Lv.0）
- Lv.0 的卡顯示 [???]，打出 3 次後升為 Lv.1（揭露完整資訊）
- 每日新詞上限 10-15 個（防疲勞）
- 解鎖更多新詞的方式：通關獎勵、商店購買、特殊事件
