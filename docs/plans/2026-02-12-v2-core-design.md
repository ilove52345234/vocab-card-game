# Vocab Card Game v2 — 核心設計文件

> 基於 Balatro 設計分析的全新設計，學習融入遊戲機制本身。

---

## 一、設計哲學

```
極簡的戰鬥驗證 + 複雜的 Build 構築 + 學習 = 遊戲操作本身
```

- 得分 = 籌碼 × 倍率（全遊戲只需理解這一個公式）
- 學習不是考試，而是**遊戲機制的一部分**
- 玩家以為自己在組牌型打分數，其實在認識單字、學詞性、記分類

## 二、超級概念

### 2.1 主概念：單字卡組合

- 類比撲克牌型，用單字的屬性組合出「牌型」得分
- 人人都認識英文字母，零門檻
- 「組合英文單字」比「背單字」有趣得多

### 2.2 雙維度系統

| 維度 | 類比撲克 | 數量 | 視覺呈現 |
|------|---------|------|---------|
| **元素（Element）** | 花色 | 5 種 | 卡牌底色/邊框顏色+圖示 |
| **詞性（Part of Speech）** | 點數 | 4 種 | 卡牌頂部標記 |

**五大元素：**

| 元素 | 顏色 | 圖示 | 涵蓋詞彙 |
|------|------|------|---------|
| 🌿 Life | 綠 | 葉子 | 動物、植物、食物、身體 |
| 🔥 Force | 紅 | 火焰 | 動作、運動、武器、工具 |
| 💧 Mind | 藍 | 水滴 | 情感、感官、溝通、藝術 |
| ⚙️ Matter | 灰 | 齒輪 | 物品、建築、材料、科技 |
| ✨ Abstract | 紫 | 星星 | 數字、時間、形狀、概念 |

**四大詞性：**

| 詞性 | 標記 | 佔比 |
|------|------|------|
| 名詞（Noun） | N | ~40% |
| 動詞（Verb） | V | ~25% |
| 形容詞（Adjective） | A | ~20% |
| 副詞/其他（Adverb+） | D | ~15% |

## 三、牌型系統

### 3.1 基礎規則

| 項目 | 數值 | 設計邏輯 |
|------|------|---------|
| 每關出牌次數 | 4 | 和 Balatro 一致 |
| 每關棄牌次數 | 3 | 和 Balatro 一致 |
| 手牌上限 | 8 | 9張會質變（某些牌型必定成立），所以定8 |
| 最多打出 | 5 張 | 和 Balatro 一致 |
| 初始牌組 | 52 張 | 基礎池，5元素×4詞性均勻分佈 |

### 3.2 牌型定義

| 牌型 | 條件 | 類比撲克 | 教育價值 |
|------|------|---------|---------|
| **單字 (Word)** | 打出 1 張 | 高牌 | 基礎認字 |
| **同性對 (PoS Pair)** | 2 張同詞性 | 對子 | 認識詞性 |
| **同族對 (Elem Pair)** | 2 張同元素 | - | 認識主題分類 |
| **三同性 (PoS Triple)** | 3 張同詞性 | 三條 | 強化詞性意識 |
| **語序鏈 (Grammar Chain)** | 3+ 張按語法順序 | 順子 | 學英文語序 |
| **同族三 (Elem Triple)** | 3 張同元素 | - | 主題詞彙群 |
| **滿堂 (Full House)** | 3張同詞性 + 2張同詞性 | 葫蘆 | 雙重詞性辨識 |
| **同族花 (Elem Flush)** | 5 張同元素 | 同花 | 深度主題學習 |
| **全同性 (PoS Flush)** | 5 張同詞性 | - | 詞性精通 |
| **語序同族 (Grammar Flush)** | 語序鏈 + 全同元素 | 同花順 | 終極組合 |

### 3.3 語序鏈規則

英文基本語法順序：**形容詞 → 名詞 → 動詞 → 副詞**（A → N → V → D）

合法的語序鏈範例：
- A → N → V（3張：beautiful → cat → run）
- N → V → D（3張：dog → eat → quickly）
- A → N → V → D（4張：big → house → stand → firmly）
- A → A → N → V → D（5張，重複詞性允許但必須維持順序）

規則：
- 最少 3 張
- 詞性必須按 A → N → V → D 順序排列（可跳過，如 A → V 合法）
- 同詞性可以連續出現（如 A → A → N 合法）
- 逆序不合法

### 3.4 牌型基礎分數

| 牌型 | 基礎籌碼 | 籌碼成長/級 | 基礎倍率 | 倍率成長/級 |
|------|---------|-----------|---------|-----------|
| 單字 | 5 | +10 | 1 | +1 |
| 同性對 | 10 | +15 | 2 | +1 |
| 同族對 | 15 | +15 | 2 | +1 |
| 三同性 | 30 | +20 | 3 | +2 |
| 語序鏈 | 30 | +30 | 4 | +3 |
| 同族三 | 35 | +20 | 3 | +2 |
| 滿堂 | 40 | +25 | 4 | +2 |
| 同族花 | 50 | +30 | 5 | +3 |
| 全同性 | 60 | +35 | 6 | +3 |
| 語序同族 | 100 | +40 | 8 | +4 |

> 語序鏈的倍率成長高於同族三，因為後期調整牌組成順子結構比同元素更難。

### 3.5 得分公式

```
得分 = (牌型基礎籌碼 + Σ打出卡牌的籌碼) × (牌型基礎倍率 + Σ加算倍率) × Π乘算倍率
```

每張單字卡自帶籌碼（基於字母數）：
- 3 字母：+3 籌碼
- 4 字母：+4 籌碼
- 5 字母：+5 籌碼
- 6 字母：+7 籌碼
- 7+ 字母：+10 籌碼

> 長單字更難記但給更多籌碼，內在動機驅動學習難字。

## 四、學習系統（融入局內）

### 4.1 核心原則

```
學習 = 遊戲操作本身，不是額外的考試
```

- 從來不彈出「請選擇正確答案」的測驗畫面
- 學習透過遊戲機制自然發生
- 玩家有動機學習，因為學會的卡牌更強

### 4.2 卡牌資訊顯示

手牌只顯示英文，不顯示中文——打出時結算動畫揭露中文+發音。

| 學習等級 | 手牌顯示 | 答題要求 | 答題形式 | 限時 | 籌碼加成 |
|---------|---------|--------|--------|------|---------|
| **Lv.0 未知** | [???] + 元素 + 詞性 | 必答 | 4選1（看英文選中文） | 3秒 | ×0.5 |
| **Lv.1 識讀** | 英文 + 元素 + 詞性 | 必答 | 2選1（看英文選中文） | 2秒 | ×1.0 |
| **Lv.2 聽力** | 英文 + 元素 + 詞性 | 必答 | 2選1（聽發音選單字） | 2.5秒 | ×1.2 |
| **Lv.3 拼字** | 英文 + 元素 + 詞性 | 必答 | 快速拼字（填空） | 4秒 | ×1.5 |
| **Lv.4 精通** | 完整資訊 + 發光 | **免答** | 自動成功 | 0秒 | ×1.5 + 版本效果 |

> Lv.4 精通的卡免答 = 遊戲自然加速。越多精通卡，節奏越快越爽。

### 4.3 答題與懲罰機制（賭狗設計）

**出牌流程：**
```
選好牌 → 點擊出牌 → 每張卡依序答題 → 牌型結算 → 得分
```

**答對：** 卡牌正常計分（全額籌碼 + 計入牌型）

**答錯：** 卡牌「受損」（半額籌碼，仍計入牌型），然後出現三個選擇：

| 選擇 | 成本 | 結果 |
|------|------|------|
| **接受** | 免費 | 該卡 50% 籌碼，繼續 |
| **花錢重答** | $2 | 新題目重答一次。答對=全額，再錯=25% |
| **賭一把** | 免費 | 50% 機率全額恢復 / 50% 機率該卡 0 籌碼 |

**關鍵：無論答對答錯，卡牌都計入牌型。** 同族花不會因為答錯降級為同族三，只是分數降低。

**連錯兜底機制：**

| 連錯數 | 觸發效果 |
|--------|--------|
| 同局連錯 3 題 | 「別氣餒！」提示 + 後續答題降為 2 選 1 |
| 同局連錯 5 題 | 本關剩餘答題全部變 2 選 1 |
| 單局全對 | 獎勵 $5 +「完美學習」標記 |

**賭狗心理設計：**
- 答錯後的三選一本身就是**風險決策**——接受損失？花錢買機會？賭一把？
- 「花錢重答」驅動**沉沒成本**——已經花了 $4，這局不能放棄
- 「賭一把」的 50/50 帶來**腎上腺素**——贏了超爽，輸了也不至於崩盤
- 經濟系統聯動：金幣要存利息還是留著當重答保險？

### 4.4 學習升級方式

| 升級路徑 | 觸發方式 |
|---------|--------|
| Lv.0 → Lv.1 | 打出此卡 3 次（每次結算動畫揭露英文+中文+發音） |
| Lv.1 → Lv.2 | 在商店花 $3 做聽力練習（聽發音確認） |
| Lv.2 → Lv.3 | 在商店花 $5 做拼字練習（拼對單字） |
| Lv.3 → Lv.4 | 累計答對此卡 20 次（自然精通，不需額外花費） |

### 4.5 遺忘曲線

| 等級 | 複習間隔 | 退化條件 | 退化效果 |
|------|---------|---------|---------|
| Lv.0 | - | - | - |
| Lv.1 | 1 天 | 1天內未在任何局中打出 | → Lv.0 |
| Lv.2 | 3 天 | 3天內未打出 | → Lv.1 |
| Lv.3 | 7 天 | 7天內未打出 | → Lv.2 |
| Lv.4 | 永久 | 不退化 | - |

> 退化 = 答題重新出現 + 籌碼降低。動機：持續使用卡牌保持等級。

### 4.6 新單字解鎖

- 每場遊戲開始時，從「未學習池」中自動加入 2-3 張新卡（Lv.0）
- Lv.0 的卡顯示 [???]，打出 3 次後升為 Lv.1
- 每日新詞上限 10-15 個（防疲勞）
- 解鎖更多新詞的方式：通關獎勵、商店購買、特殊事件

## 六、經濟系統

### 6.1 設計哲學（麻將模型）

```
學習 ≠ 額外花費    學習 = Build 的過程本身
失敗 ≠ 歸零        失敗 = Build 降級（清一色 → 平胡）
```

- 金幣圍繞「養牌」展開——升級卡牌既是學習也是 Build
- 養牌是最划算的投資：Lv.4 精通卡免答 + ×1.5 加成，長期回報最高
- 玩家永遠有牌可打，只是分高分低（牌型降級 + 答題降級）

| $5-9 | +$1 | $1 |
| $10-14 | +$2 | $3 |
| $15-19 | +$3 | $6 |
| $20-24 | +$4 | $10 |
| $25+ | +$5（上限） | $15 |

### 6.5 麻將式降級模型

**牌型降級（摸牌不順）：**

| 目標 | 實際能組的 | 體感 |
|------|----------|------|
| 同族花（5 Life） | 同族三（3 Life） | 沒湊到大牌，但也有分 |
| 語序同族 | 語序鏈 | 退而求其次 |
| 全同性（5 名詞） | 三同性（3 名詞） | 還算合理 |

**答題降級（學習不夠）：**

| 情境 | 得分影響 | 經濟影響 |
|------|---------|---------|
| 5 張全對 | 全額 + 全對獎 $5 | 最佳 |
| 3 對 2 錯（接受） | ~80% 得分 | 少賺但穩定 |
| 2 錯（花 $2 重答） | 可能挽回 90%+ | -$4 保分 |
| 2 錯（賭一把） | 50% 全額 / 50% 歸零 | 賭狗路線 |

> 養牌不夠 = 你的「清一色」打折扣。永遠有分拿，只是分高分低。

### 7.1 商店結構

每過一關（小盲注/大盲注/Boss 後）進入商店：

```
┌─────────────────────────────────────────────────┐
│  商店                                            │
│                                                   │
│  ── 商品區（5格）──                               │
│  [語感卡]  [語感卡]  [教材卡]  [詞綴卡]  [詞綴卡] │
│   $6        $5        $4        $3        $2      │
│                                                   │
│  ── 養牌區（3格，永遠存在）──                      │
│  [cat→Lv.2 $3]  [run→Lv.2 $3]  [big→Lv.3 $5]    │
│                                                   │
│  [重擲 $5]  [卡包 $4]              [下一關 →]     │
│                                                   │
│  Boss 關後追加：                                   │
│  [速成班 $10]  [精讀班 $10]                       │
└─────────────────────────────────────────────────┘
```

### 7.2 商品區（5 格）

| 商品 | 售價 | 出現率 |
|------|------|--------|
| 語感卡（普通） | $4-5 | 常見 |
| 語感卡（稀有） | $6-7 | 較少 |
| 語感卡（傳說） | $8 | 極少 |
| 教材卡 | $3-6 | 常見 |
| 詞綴卡 | $2-4 | 常見 |

每關隨機刷新。重擲費用：$5 → $10 → $15（遞增）。

### 7.3 養牌區（3 格，不受重擲影響）

系統從牌組中自動挑出 3 張可升級的卡展示：

| 升級 | 費用 | 測驗方式 |
|------|------|---------|
| Lv.0 → Lv.1 | 免費 | 打出 3 次自動升級 |
| Lv.1 → Lv.2 | $3 | 聽力小測驗（聽發音確認） |
| Lv.2 → Lv.3 | $5 | 拼字小測驗（拼對單字） |
| Lv.3 → Lv.4 | 免費 | 累計答對 20 次自動升級 |

> 養牌區永遠存在，不佔商品格，不被重擲刷掉。確保每次進商店，養牌都是最顯眼的選項。

### 7.4 卡包（偶爾出現）

### 7.5 經濟節奏
