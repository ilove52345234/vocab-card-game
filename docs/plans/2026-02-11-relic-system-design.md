# 遺物系統設計 v2

*建立日期：2026-02-11*
*更新日期：2026-02-11（v2：加入字根遺物，三層構詞學體系）*

---

## 一、設計目標

1. **跨戰鬥策略層**：遺物是整個 run 的被動效果，影響構築方向
2. **三層構詞學教學**：字首（方向）、字尾（詞性）、字根（核心語意），使用遺物 = 學習構詞學
3. **維度引導**：部分遺物強化特定維度，鼓勵玩家發展專精路線
4. **詞庫連動**：字根遺物與牌組中含該字根的卡牌產生額外加成

---

## 二、核心概念：三層構詞學遺物

遺物名稱來自英文最常用的 Prefixes、Suffixes 和 Word Roots。

| 類型 | 語言功能 | 遊戲角色 | 數量 |
|------|---------|---------|------|
| **字首 (Prefix)** | 改變方向/語意 | 觸發型：改變某件事的運作方式 | 6 |
| **字尾 (Suffix)** | 改變詞性/類別 | 增幅型：強化現有效果 | 6 |
| **字根 (Root)** | 核心含義 | 基石型：持續被動 + 詞庫連動 | 6 |

### 學習循環

```
獲得遺物 → 看到字首/字尾/字根 + 語意 → 戰鬥中反覆觸發 → 記住語意
   ↓
遇到含該構詞元素的新單字 → 能自行推測意思 → 詞彙量自然擴展
```

### 字根遺物的獨特機制：詞庫連動

字根遺物擁有字首字尾沒有的特性。持有某字根遺物時，牌組中**含有該字根的卡牌**獲得額外 +10% 效果加成。

- 範例：持有 `struct`（建造）遺物 → construct, structure, instruct, destroy 等卡 +10%
- 玩家自然會關注「哪些字含有這個字根」→ 培養拆字能力
- MVP 詞庫 30 字覆蓋率有限，隨詞庫擴展此機制價值遞增

## 三、遺物分類

| 類別 | 比例 | 遊戲角色 |
|------|------|---------|
| **觸發型（字首）** | ~33% | 改變規則、觸發特殊效果 |
| **增幅型（字尾）** | ~33% | 強化數值、維度加成 |
| **基石型（字根）** | ~33% | 持續被動 + 詞庫連動 |

## 七、遺物與標準值的關係

遺物效果不納入卡牌標準值計算。遺物是「構築獎勵」——選對遺物讓牌組更強，但不影響單卡平衡。

**遺物強度基準：**
- 字尾遺物（增幅型）：約等於每場 +3~5 點標準值（穩定但幅度小）
- 字首遺物（觸發型）：約等於每場 +5~10 點標準值（條件觸發但效果大）
- 字根遺物（基石型）：被動約 +3~5 點 + 詞庫連動約 +2~8 點（隨詞庫規模成長）
