# 遺物系統設計

*建立日期：2026-02-11*

---

## 一、設計目標

1. **跨戰鬥策略層**：遺物是整個 run 的被動效果，影響構築方向
2. **與學習系統深度結合**：遺物以英文字首字尾命名，使用遺物 = 學習字首字尾
3. **維度引導**：部分遺物強化特定維度，鼓勵玩家發展專精路線

---

## 二、核心概念：字首字尾遺物

遺物名稱來自英文最常用的 Prefixes 和 Suffixes。每個遺物：
- **名稱**包含字首/字尾
- **效果**與字首/字尾的語意掛鉤
- **描述**列出相關單字範例

**學習循環：**
```
獲得遺物 → 看到字首/字尾 + 語意 → 戰鬥中反覆使用 → 記住語意
   ↓
遇到含該字首/字尾的新單字 → 能自行推測意思 → 詞彙量自然擴展
```

**未來擴展**：詞庫達 1000+ 字時，遺物可額外給予「含有該字首字尾的單字卡 +10% 效果」。

## 三、遺物分類

| 類別 | 比例 | 獲取來源 | 設計意圖 |
|------|------|---------|---------|
| **戰鬥遺物** | ~40% | 精英掉落、寶箱 | 直接影響戰鬥數值 |
| **維度遺物** | ~20% | 精英掉落 | 強化特定維度(Strike/Guard/Boost/Warp) |
| **學習遺物** | ~30% | 休息站(學新字時)、寶箱 | 影響答題和學習效果 |
| **休息站遺物** | ~10% | 寶箱（稀有） | 為休息站增加新選項 |

## 四、獲取途徑

| 來源 | 稀有度 | 數量 |
|------|--------|------|
| 精英掉落 | Common-Uncommon | 每隻精英掉 1 個 |
| 寶箱（第 9 步） | Common-Rare | 1 個 |
| 章節過渡（打敗 Boss 後） | 3 選 1 | Common-Rare |
| 休息站學新字時（低機率） | Common | ~20% 機率額外獲得 |

## 六、效果與語意的對應邏輯

| 字首/字尾 | 語意 | 效果對應 |
|----------|------|---------|
| -tion | 狀態/動作 | 「知識的狀態」→ 更多答題時間 |
| -ness | 性質 | 「堅韌的性質」→ 開場格擋 |
| -er | 做...的人 | 「攻擊者」→ 首張攻擊加傷 |
| re- | 再次 | 「再次充能」→ 能量回復 |
| -ly | 方式 | 「明智地回答」→ 答對回血 |
| -ful | 充滿 | 「充滿防護」→ Guard +2 |
| -less | 沒有 | 「無可阻擋」→ Strike +2 |
| -ment | 結果 | 「增長的結果」→ 額外抽牌 |
| -able | 能夠 | 「有能力做到」→ 效果 +30% |
| pre- | 之前 | 「預先準備」→ 費用 -1 |
| un- | 否定/反轉 | 「解鎖潛能」→ 新休息站選項 |
| dis- | 離開/消除 | 「消除死亡」→ HP 歸零復活 |

## 七、遺物與標準值的關係

遺物效果不納入卡牌標準值計算。遺物是「構築獎勵」——選對遺物讓牌組更強，但不影響單卡平衡。

**遺物強度基準：**
- Common：約等於每場 +3~5 點標準值
- Uncommon：約等於每場 +5~8 點標準值
- Rare：約等於每場 +8~15 點標準值或獨特機制
