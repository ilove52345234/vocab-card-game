# 詞族進化系統設計

*建立日期：2026-02-11*
*更新日期：2026-02-11（v2：從單純升級重構為詞族進化系統）*

---

## 一、核心概念：進化 = 詞族擴展

卡牌不只是「升級變強」，而是在熟練度達標後可以**進化為同詞族的不同單字**，成為一張全新的卡牌。這直接對應英語中的 Word Family（詞族）概念。

**進化不是上位替代**——進化後的卡牌有不同的費用、效果、維度，甚至元素。它是一張**全新的卡**，只是與原卡有詞源上的親緣關係。

## 二、卡牌獲取方式全覽

| 方式 | 地點 | 代價 | 選擇權 | 學習行為 |
|------|------|------|--------|---------|
| **起始牌組** | Run 開始 | 無 | 無（固定 5 張基礎字） | 無 |
| **戰鬥獎勵** | 戰鬥後 | 無 | 3 選 1 | 需通過首次答題 |
| **休息站學字** | 休息站 | 放棄回血/升級 | 3 選 1（未學池） | 需通過首次答題 |
| **詞族進化** | 休息站或書房 | 熟練度歸零 | 選擇進化方向 | 需通過新字首次答題 |
| **書房預習** | 書房 | 8 學習點數 | 自由瀏覽選擇 | 需通過首次答題 |
| **未知房事件** | 未知房 | 看事件內容 | 接受/拒絕 | 視事件而定 |
| **Boss 獎勵** | 章節結束 | 無 | 3 選 1（稀有字） | 需通過首次答題 |

**核心原則**：每一張新卡進入牌組前，玩家都必須通過至少一次答題。沒有任何方式可以跳過學習直接獲卡。

## 三、熟練度升級（休息站）

### 休息站三選一

| 選項 | 效果 | 與學習的關係 |
|------|------|------------|
| **回血** | 回復 30% 最大 HP | 純生存選擇 |
| **升級卡牌** | 選 1 張卡，立即答題 3 次 | 直接提升熟練度（可觸發進化） |
| **學新字** | 從未學池 3 選 1，首次答題 | 解鎖新卡牌加入牌組 |

### 升級流程

1. 選擇「升級卡牌」
2. 從當前牌組中選 1 張卡
3. 連續進行 3 次答題（難度依當前熟練度）
4. 依答對次數計算結果

### 答題結果

| 答對次數 | 結果 | 說明 |
|---------|------|------|
| 3/3 | 熟練度 +1 | 完美複習，完整升級 |
| 2/3 | 熟練度 +1，複習間隔不延長 | 升級但需要更快複習 |
| 1/3 | 熟練度不變，重置複習計時器 | 沒有退步，但重新計時 |
| 0/3 | 熟練度 -1 | 完全忘記，退步 |

### 答題模式

答題模式依當前熟練度決定（與戰鬥中答題一致）：
- Lv.1-2：識讀模式
- Lv.3-4：識讀 + 聽力混合
- Lv.5-6：聽力 + 拼字混合
- Lv.7+：不可再升級（已達上限或已深化）

### 升級對卡牌的影響

熟練度提升後，卡牌獲得兩項增益：

1. **效果倍率提升**（依標準值體系）：
   - Lv.1-2：×1.0
   - Lv.3-4：×1.1
   - Lv.5-6：×1.2
   - Lv.7：×1.3
   - Lv.8：×1.4（需深化解鎖）
   - Lv.9：×1.5（需深化解鎖）

2. **答題機率降低**（戰鬥中更快打出）：
   - Lv.1：100% 需答題
   - Lv.3：60%
   - Lv.5：20%
   - Lv.7：0%（自動發動）
   - Lv.8-9：0%（自動發動，深化專精）

## 四、詞族進化機制

### 觸發條件

- 卡牌達到 **Lv.5** 熟練度
- 可在**休息站**（選「升級卡牌」時出現進化選項）或**書房**（進化服務）執行

### 進化選項：三叉選擇

當一張 Lv.5 卡進行升級時，玩家面對三個選擇：

```
         protect (Lv.5)
        /       |       \
  詞族進化    同字深化    繼續升級
  (新卡牌)  (極致專精)  (穩定成長)
```

### 選項 A：詞族進化

- 進化為同詞族的**不同單字**
- 新卡是全新的卡牌（不同費用、效果、可能不同維度/元素）
- **熟練度歸零**（Lv.0 → 需重新學習）
- 需通過新字的首次答題才能完成進化
- 舊卡被**替換**（不保留）

### 選項 B：同字深化

- 同一個字，等級上限從 Lv.7 提升至 **Lv.9**
- 熟練度不歸零，保持在 Lv.5
- 基礎效果不變，但可以繼續升級到更高倍率
- 代價：需花費 **6 學習點數**（書房）或放棄一次休息站機會

### 選項 C：繼續升級

- 正常升級流程，嘗試提升到 Lv.6
- 無額外代價，無額外收益
- 穩定路線

### 進化的策略權衡

| 面向 | 進化前 (protect Lv.5) | 詞族進化 (protection Lv.0) | 同字深化 (protect+ Lv.5) |
|------|----------------------|--------------------------|------------------------|
| 基礎效果 | 10 格擋 | 17 格擋 | 10 格擋 |
| 熟練度倍率 | ×1.2 | ×1.0 | ×1.2（可升至 ×1.5） |
| 實際效果 | 12 格擋 | 17 格擋 | 12 → 最高 15 格擋 |
| 答題機率 | 20% | 100% | 20% → 0% |
| 費用 | 1 費 | 2 費 | 1 費 |
| 維度 | Guard | Guard（可能不同） | Guard |
| 短期戰力 | 穩定 | 下降（要重新學） | 穩定 |
| 長期潛力 | 上限 Lv.7 | 全新上限 Lv.7 | 上限 Lv.9 |

## 五、詞族進化範例

### 範例 1：protect 詞族

```
protect (動詞, Guard, 1費: 10格擋)
  ├→ protection (名詞, Guard, 2費: 17格擋 + 持續1回合)
  │     更重型的防禦，費用更高
  ├→ protective (形容詞, Boost, 1費: 全體+3格擋)
  │     從個人防禦轉為團隊增益，維度改變
  └→ protector (名詞, Strike, 2費: 8傷害 + 8格擋)
        攻守兼備，維度改變為 Strike
```

### 範例 2：run 詞族

```
run (動詞, Strike, 1費: 12傷害)
  ├→ runner (名詞, Strike, 1費: 8傷害 + 抽1牌)
  │     傷害降低但獲得抽牌，偏向連段
  └→ running (形容詞/動名詞, Boost, 0費: +2力量本回合)
        從直傷轉為增益，低費快速
```

### 範例 3：happy 詞族

```
happy (形容詞, Boost, 1費: +3力量 1回合)
  ├→ happiness (名詞, Warp, 2費: 回復5HP + 抽1牌)
  │     從增益轉為回復/資源，維度改變
  └→ happily (副詞, Boost, 0費: 下一張卡效果+20%)
        低費增幅卡，維度不變
```

## 六、與 SM-2 的整合

所有答題結果（升級、進化、戰鬥）統一寫入 SM-2 演算法：
- 答對視為 quality 4-5（依速度）
- 答錯視為 quality 1-2
- 影響 easeFactor 和 nextReviewTime

進化產生的新字以 Lv.0 全新記錄進入 SM-2 系統，開始獨立的記憶曲線。

## 七、策略考量

### 何時進化？

- **牌組缺少某維度**：透過進化將 Guard 卡轉為 Strike 卡填補空缺
- **協同需求**：進化後的卡可能產生新的資源媒介標籤，開啟新的協同鏈
- **挑戰自我**：進化後熟練度歸零 = 戰鬥中 100% 需答題，但知識共振加成（Lv.1 跟在 Lv.5+ 後 +30%）可以彌補

### 何時深化？

- **核心卡牌**：最常用的卡深化到 Lv.9，倍率 ×1.5 + 永久免答題
- **知識共振觸發器**：Lv.8-9 的卡可以穩定觸發洞察標記和知識共振加成

### 風險

- 進化後短期變弱，如果在 Boss 前進化可能致命
- 但這個風險驅動學習——進化後會積極練習新字以盡快提升熟練度
